## Frontend Rules

- Use Tailwind for all styling — no inline styles, no CSS modules
- Components must be functional — no class components
- Always add aria-labels to interactive elements
- Use React Query for server state, Zustand for client state
- File naming: PascalCase for components, camelCase for hooks
- Export components as named exports, not default exports
